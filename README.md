## 📌 Task 1: Titanic Survival Prediction

### 📖 Description

Predict whether a passenger survived the Titanic disaster using machine learning techniques.

### ⚙️ Key Steps

* Data Cleaning & Preprocessing
* Feature Engineering (FamilySize, Title, AgeGroup, etc.)
* Exploratory Data Analysis (EDA)
* Model Building (Logistic Regression, Random Forest)

### 📊 Key Insights

* Women had higher survival rates than men
* First-class passengers had higher survival rates
* Children were more likely to survive
* Small families had better survival chances
* Passengers traveling alone had lower survival rates
